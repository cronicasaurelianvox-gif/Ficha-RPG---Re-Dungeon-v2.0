🔑 DETALHE IMPORTANTE: Novos IDs
════════════════════════════════════════════════════════════════

POR QUÊ gerar novos IDs?

  ❌ ERRADO: Usar os mesmos IDs
     ├─ Possível conflito se editar original e clone simultaneamente
     ├─ Banco de dados pode ficar confuso
     └─ Comportamento imprevisível

  ✅ CORRETO: Gerar novos IDs únicos
     ├─ Cada item tem ID único no clone
     ├─ Sem conflito com original
     └─ Comportamento previsível

EXEMPLO:

  Item Original:
  {
    id: "item_1234567_abc123def",
    nome: "Espada de Ferro",
    quantidade: 2
  }

  Item Clonado:
  {
    id: "item_1711824000456_xyz789uvw",  ← NOVO ID!
    nome: "Espada de Ferro",
    quantidade: 2
  }
